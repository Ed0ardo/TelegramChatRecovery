Telegram Chat Recovery Tool

This tool allows you to recover a Telegram chat from a backup by converting it into a WhatsApp-compatible format. You can upload multiple backup files (JSON or HTML), and the tool will combine their contents into a single TXT file. This file can then be reimported into Telegram using the import feature from WhatsApp. JSON format is preferred. All files must be of the same format (either all JSON or all HTML).

Instructions:

  1. Export your Telegram chat as JSON or HTML (preferably JSON).
  2. Extract the backup ZIP file provided by Telegram.
  3. Upload the JSON or HTML files using the button above. You can select multiple files at once.
  4. The tool will combine the converted content into a single TXT file.
  5. Save the TXT file in the same folder as your Telegram backup.
  6. Select all files in the backup folder and create a new ZIP file.
  7. Send the ZIP file to yourself on WhatsApp.
  8. On WhatsApp, select the ZIP file, share it to Telegram, and choose the chat where you want to import the conversation.
  9. Click "Import" to complete the process.
